# 📝 TodoApp-FastAPI

A simple and efficient Todo application built with **FastAPI**, designed to help users manage tasks with ease. This project demonstrates clean architecture, RESTful API design, and integration with modern Python tooling.

## 🚀 Features

- Create, read, update, and delete todos (CRUD)
- Fast and asynchronous API powered by FastAPI
- SQLite or PostgreSQL database support via SQLAlchemy
- Pydantic models for data validation
- JWT-based authentication

## 📦 Tech Stack

- **FastAPI** – high-performance web framework
- **SQLAlchemy** – ORM for database operations
- **Uvicorn** – ASGI server
- **Pydantic** – data validation
- **Alembic** – database migrations (optional)
- **Python 3.10+**

- All Of the requirements is listed in the requirements.txt file.
